# Сортировка файлов по дате

Программа на PySide6: анализирует файлы в выбранной папке по дате создания или изменения и перемещает их в подпапки с именами дат.

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Функции

1. **Выбор папки** — кнопка «Выбрать папку…», под ней отображается путь и количество найденных файлов.
2. **Подпапки** — чекбокс «Включая подпапки» для учёта файлов из вложенных папок.
3. **Формат имени папки** — выбор формата: `ГГГГ_ММ_ДД` (по умолчанию) или `ДД_ММ_ГГГГ`.
4. **Сортировка внутри папки с датой** — два чекбокса:
   - **По расширению** — внутри папки даты создаются подпапки JPG, PNG, TXT и т.д.;
   - **По размеру** — подпапки вида «1 КБ», «10 МБ», «1 ГБ» и т.д.
   Оба варианта можно включить: тогда структура будет «дата → расширение → размер».
5. **Перемещение** — кнопка «Переместить файлы» запускает сортировку.
6. **Прогресс и отчёт** — прогресс-бар и текстовый отчёт о выполнении.

Папки с датами создаются внутри выбранной папки, файлы перемещаются по дате создания (при недоступности — по дате изменения). При совпадении имён к имени добавляется суффикс `_001`, `_002` и т.д.
